# ATRI Project
## 使用
这是一个基于mirai框架，graia SDK的qqbot  
您只需要在cfg.json文件里设置bot参数即可

## 功能
目前实现的功能有
(重构后以下功能皆暂未实现，请等待后续更新)
- 以图搜图  
- 以图搜番  
- 查询疫情
- 查询天气 
- 摸头
- 基础闲聊
- ~~随机涩图~~
